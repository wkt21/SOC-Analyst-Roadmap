# SOC Analyst Labs

Hands-on labs to reinforce SOC analyst skills across networking, threat intelligence, SIEM, EDR, and more.

---

## Networking Labs
- Analyze PCAPs for suspicious traffic.
- Identify DNS tunneling patterns.
- Detect beaconing using Zeek logs.

---

## Operating System Labs
- Investigate Windows Event IDs for authentication anomalies.
- Parse Linux auth logs for brute-force attempts.
- Build PowerShell scripts for triage automation.

---

## Threat Intelligence Labs
- Enrich IOCs using VirusTotal and Shodan.
- Map a known threat actor to ATT&CK techniques.
- Analyze phishing domains for lifecycle patterns.

---

## SOC Operations Labs
- Build Splunk dashboards for login events.
- Write correlation rules for brute-force detection.
- Investigate EDR alerts and isolate compromised hosts.

---

## Vulnerability Monitoring Labs
- Run Nessus scans and validate findings.
- Compare system configs against CIS benchmarks.
- Detect unauthorized configuration drift.

---

## Identity & Access Labs
- Analyze Azure AD sign-in logs.
- Detect impossible travel events.
- Investigate privilege escalation anomalies.

---

## Infrastructure Monitoring Labs
- Review firewall logs for segmentation violations.
- Inspect TLS handshake failures.
- Analyze VPN logs for suspicious access.

---

## Advanced SOC Labs
- Deploy a honeypot and analyze interactions.
- Conduct purple team simulations.
- Map adversary behavior to ATT&CK.

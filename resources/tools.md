# SOC Tools Reference

A curated list of tools commonly used across SOC operations, threat intelligence, vulnerability management, and monitoring.

---

## Networking & Traffic Analysis
- **Wireshark** — Packet capture and protocol analysis.
- **Zeek** — Network security monitoring and log generation.
- **tcpdump** — CLI packet capture.
- **Nmap** — Network scanning and host discovery.

---

## SIEM Platforms
- **Splunk** — Enterprise log management and correlation.
- **ELK Stack** — Elasticsearch, Logstash, Kibana.
- **IBM QRadar** — SIEM with advanced correlation.
- **Microsoft Sentinel** — Cloud-native SIEM/SOAR.

---

## EDR / XDR
- **CrowdStrike Falcon**
- **SentinelOne**
- **Microsoft Defender for Endpoint**
- **Sophos Intercept X**

## Threat Intelligence & OSINT
- **VirusTotal** — Hash, domain, and file analysis.
- **Hybrid Analysis** — Sandbox detonation.
- **Shodan** — Internet-facing asset discovery.
- **Censys** — Certificate and host enumeration.
- **Maltego** — Graph-based intelligence mapping.
- **SpiderFoot** — Automated OSINT collection.

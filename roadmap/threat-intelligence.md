# Threat Intelligence

Threat intelligence enables SOC analysts to understand adversary behavior, track indicators, and anticipate attacks.

---

## OSINT Workflows
- WHOIS, DNS lookups, passive DNS.
- IP reputation checks.
- Hash analysis (VirusTotal, Hybrid Analysis).
- Email header analysis.

### Tools
- Maltego
- SpiderFoot
- Shodan
- Censys

---

## Threat Hunting
- Identify suspicious patterns in logs.
- Map behaviors to ATT&CK TTPs.
- Build hypotheses and test them with data.
- Prioritize alerts based on severity and context.

### Labs
- Hunt for persistence mechanisms on a Windows host.
- Identify lateral movement using event logs.

---

## IOC Analysis
- **IPs** — GeoIP, ASN, reputation.
- **Hashes** — File behavior, sandboxing.
- **Domains** — Age, registrar, DNS history.

### Labs
- Build an IOC enrichment script.
- Analyze a phishing domain lifecycle.

---

## Learning Objectives
- Build repeatable OSINT workflows.
- Map threats to ATT&CK techniques.
- Enrich and validate indicators effectively.

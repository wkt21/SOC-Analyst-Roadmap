# Core Skills

Building a strong foundation in networking, operating systems, and cybersecurity fundamentals is essential for any SOC analyst. These skills support log analysis, threat detection, and incident response.

---

## Networking Fundamentals
- **TCP/IP** — Understand packet structure, ports, protocols.
- **DNS** — Query flow, record types, common attack patterns.
- **DHCP** — Lease process, rogue DHCP detection.
- **Subnetting** — CIDR notation, network segmentation.
- **Network Design** — VLANs, routing, firewalls, DMZs.

### Labs
- Capture and analyze packets with Wireshark.
- Build a small virtual network and practice subnetting.
- Inspect DNS logs and identify suspicious queries.

---

## Operating Systems

### Windows
- **Active Directory** — Users, groups, GPOs, domain controllers.
- **Event Logs** — Security, System, Application.
- **Authentication Flow** — Kerberos, NTLM.
- **PowerShell** — Scripting for triage and automation.

### Linux
- **Permissions & Ownership** — chmod, chown, sudo.
- **Syslog** — Log locations, rsyslog pipeline.
- **Processes & Services** — systemd, ps, top.
- **Shell Scripting** — Bash automation for log parsing.

### Labs
- Analyze Windows Event ID patterns.
- Parse Linux auth logs for brute-force attempts.
- Write scripts to extract IOC patterns from logs.

---

## Cybersecurity Basics
- **CIA Triad** — Confidentiality, Integrity, Availability.
- **Risk Assessment** — Threats, vulnerabilities, impact.
- **Threat Frameworks**
  - MITRE ATT&CK
  - Cyber Kill Chain
  - Diamond Model

### Labs
- Map a known attack to ATT&CK techniques.
- Perform a basic risk assessment on a small environment.

---

## Learning Objectives
- Understand how networks communicate.
- Read and interpret OS logs confidently.
- Apply cybersecurity fundamentals to real SOC workflows.

# SOC Operations

SOC operations involve monitoring, triaging, investigating, and responding to security events.

---

## SIEM
- Splunk, ELK, QRadar fundamentals.
- Log ingestion pipelines.
- Correlation rules.
- Dashboard creation.

### Labs
- Build a Splunk dashboard for authentication events.
- Write a correlation rule for brute-force detection.

---

## Incident Response
- Alert triage workflow.
- Evidence collection.
- Basic forensics (memory, disk, network).
- Documentation and escalation.

### Labs
- Perform IR on a simulated malware infection.
- Extract artifacts from a memory image.

---

## EDR
- CrowdStrike, SentinelOne, Defender for Endpoint.
- Process trees and behavioral detections.
- Host isolation and remediation.

---

## Network Security Monitoring (NSM)
- Zeek logs: conn, dns, http, ssl.
- PCAP analysis with Wireshark.
- Detecting anomalies in network traffic.

### Labs
- Analyze Zeek logs for suspicious beaconing.
- Identify exfiltration patterns in PCAPs.

---

## Learning Objectives
- Confidently triage alerts.
- Use SIEM and EDR tools effectively.
- Perform basic forensic investigations.

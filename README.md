# 🎓 SOC Analyst Roadmap

A structured roadmap for building practical SOC analyst skills across core networking, operating systems, threat intelligence, SOC operations, vulnerability monitoring, IAM, infrastructure monitoring, awareness, compliance, and advanced blue-team skills.

## Roadmap

### Core Skills
- Networking
  - TCP/IP, DNS, DHCP
  - Subnetting & network design
- Operating Systems
  - Windows: AD, logs, Group Policy
  - Linux: permissions, syslog, scripting
- Cybersecurity Basics
  - CIA triad, risk assessment
  - Threat frameworks, including MITRE ATT&CK

### Threat Intelligence
- OSINT tools and workflows
- Threat hunting: TTPs, alert triage
- IOC analysis: IPs, hashes, domains

### SOC Operations
- SIEM: Splunk, ELK, QRadar; log analysis
- Incident response: alert handling, basic forensics
- EDR: CrowdStrike, SentinelOne; endpoint monitoring
- NSM: Zeek, Wireshark; traffic analysis

### Vulnerability Monitoring
- Scanning: Nessus, Qualys; result analysis
- Patching: track and verify updates
- Configurations: monitor secure baselines

### Identity & Access
- Authentication: MFA, SSO logs
- Authorization: RBAC/ABAC monitoring
- Anomalies: user behavior, brute-force detection

### Infrastructure Monitoring
- Segmentation: VLAN, firewall logs
- Zero Trust: identity and policy checks
- Encryption: TLS/SSL, VPN monitoring

### Awareness Support
- Phishing simulations and training metrics
- Incident feedback for user education

### Compliance & Policy
- Regulations: GDPR, HIPAA, PCI-DSS
- Policy: monitor security and IR adherence

### Advanced SOC Skills
- Deception: honeypots, alert analysis
- Simulation: purple teaming, ATT&CK mapping

## How to use this repo
1. Start with `roadmap/core-skills.md`.
2. Work through each section in order or by priority.
3. Add notes, lab results, and screenshots as you progress.
4. Use the templates in `templates/` for repeatable documentation.
